Daily Print


Previous Next

Présentation

Le projet Daily Print est une application web cherchant a améliorer le système de reprographie de l'IUT de Vélizy. Cette application permet aux professeurs de faire des requêtes d'impression et le service de reprographie peut visualiser ses requêtes et en indiquer le statut ajoutant ainsi un aspect de suivi de requêtes. De plus cette application générère des statistiques pour le service de reprographie. Ce projet est mené par une équipe de 4 personnes.


Technologies utilisées

Pour réaliser ce projet nous avons utilisés Bootstrap V4 associé a du AJAX / JQUERY / AngularJS pour le front-end et du PHP en guise de backend.
