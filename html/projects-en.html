Daily Print


Previous Next

Presentation

The Daily Print project is a web application seeking to improve the reprographic system from Vélizy's IUT. This application allows teachers to make print requests and the reprographic service can view its queries and indicate their status, thus adding a request tracking aspect. In addition, this application generates statistics for the service of reprography. This project is led by a team of 4 people.


Technologies used

To realize this project we used Bootstrap V4 with AJAX / JQUERY / AngularJS for the front end and PHP as a back end.
